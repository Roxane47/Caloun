== Description ==

= FRANCAIS =

Diaporamas a été conçu pour être simple d'utilisation et utilise le dernier stantard html pour afficher vos images avec les balises html5.

Il vous permet de positionner à l'aide de shortcodes autant de diaporamas que vous le souhaitez, que ce soit dans vos articles ou vos pages.

= ENGLISH =

Diaporamas was designed to be simple to use and uses the last html standard to view your images with html5 tags.

It allows you to position using shortcodes as many slide shows as you want, whether in your articles or your pages.

== Installation ==

= FRANCAIS =

1. Téléchargez le plugin via le bouton download ci dessus, et installez les fichiers sur votre serveur. Ou depuis le menu 'Extensions' du Tableau de bord de Wordpress cliquez sur ajouter et rechercher l'extension Diaporamas.
2. Activez le plugin depuis le menu de WordPress.
3. Mettez en place, à votre convenance, un Widget ou un Shortcode `[flexslider slug="exemple"]`.

= ENGLISH =

1. Download the plugin via the download button above, and install the files on your server. Or from the menu 'Plugins' of Wordpress Dashboard click on 'Add New' button and search plugin Diaporamas.2. Activate the plugin in the WordPress menu.3. Use a widget or a shortcode `[flexslider slug="sample"]`.
